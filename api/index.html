Get Started | API Documentation | Borrowed Books

The Borrowed Books API allows apps to interact directly with the books platform, and increasing the efficiency of managing large or complex borrowing books services.
Some typical use cases include: Get all Borrowed Books, Get books by category, Get books by Borrower Phone, Get books from search.

Introduction

The Borrowed Books Movie API created to give details about Borrowed Books Movies.
All the requests should be in GET / POST / PUT Method (Depends on the function).
The result will return in JSON File.
If your URL request is wrong you will forward to 200 Error page.

Get All Borrowed Books Movies

You can find all Borrowed Books movies, by entering the following URL:

https://ws-project1.herokuapp.com/getAllComingSoon Method: GET
