Event Break | API Documentation

The Event Break API allows directly interaction with the Event Break platform,
and get details about all the events.

Some typical use cases include:

Get all Events
Get all users
Get all equipment per event

The requests should be in GET / POST Method.
The result will return in JSON Object.
If your URL request is wrong you will recieve JSON Object Error.

Get Books By category

You can find Borrowed Books by their category.
You need sending parameter 'category' to the following URL:

https://borrowed-books.herokuapp.com/getBooksByCategory

Method: POST
Parameter name: category

Get Books By Category and Borrower Phone (Category/Phone)

You can find Borrowed Books with category and phone filter
For get it, you need entering the following URL:

https://borrowed-books.herokuapp.com/getBooksByCategoryAndPhone/[CATEGORY]/[PHONE]

Method: POST
Parameters name: /category/phone

Example of 'category' can be 'psychology'
Example of 'phone' can be '0587588846'
